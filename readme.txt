== Description ==

Custom Tables is a WordPress plugin that allows you to add custom tables to your WordPress database. This can be useful for storing and managing data that doesn't fit well into the default WordPress post and metadata structure. Whether you're building a complex plugin or need to store specific data for your website, Custom Tables provides a simple and efficient solution.

= Features =

- Create and manage custom tables with ease.
- Store and retrieve data in a structured way.
- Integrate custom tables into your plugin or theme.
- Enhance performance by optimizing data storage.

For more information and usage instructions, please visit the [plugin documentation](https://unknown-sudo-max.github.io/custom-tables).

== Installation ==

1. Upload the `custom-tables` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure and use custom tables as needed.
